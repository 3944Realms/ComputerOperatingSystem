# COS  使用文档

## 概述

COS 是一个基于 Kotlin 开发的支持配置化的模拟场景的模拟系统。系统提供交互式操作界面和详细的日志记录功能。

## 系统特性

- ✅ YAML 配置文件驱动
- ✅ 交互式命令行界面
- ✅ 日志记录
- ✅ 实时状态监控
